# ビリヤードシミュレータ

- 擬似的なビリヤードシミュレータです。
- 挙動は不完全です。

## 動作内容

- クリックした方向にキューボールを飛ばします。

### ボタン説明

- ```Reset```: 状態をリセットします。
- ```Demo```: 摩擦が無い状態のデモンストレーションを行います。```Reset```ボタンを押すことで解除されます。
- ```Redo```: 一手前の状態に戻すことができます。
